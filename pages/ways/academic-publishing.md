Steps to publish a paper open source:

*Phase 1*
1. Go to [Zenodo](https://zenodo.org/)
2. Click new upload
3. Click button "get a DOI now"
4. Add DOI to markdown sourcefile
5. Use [Pandoc](https://pandoc.org/) to convert paper to PDF
6. Open PDF and use tools to save as PDF/A

*Phase 2*
1. Add file and record to Zenodo
2. Add file and record to [KCWorks](https://hcommons.org/members/steinea/)
3. Add file and record to [ResearchGate](https://www.researchgate.net/profile/Eric-Stein-4)

*Phase 3*
1. Add record to [ORCID](https://orcid.org/0000-0003-4131-2695)
2. Add record to [Zotero](https://www.zotero.org/steinea)
3. Add record to [FigShare](https://figshare.com/authors/Eric_Stein/21583022)
4. Add record to [Google Scholar](https://scholar.google.com/citations?view_op=list_works&hl=en&hl=en&user=7qubWGsAAAAJ&sortby=pubdate)
5. Add record to [LinkedIn](https://www.linkedin.com/in/steinea/details/publications/)
6. Import reference to [Web of Science](https://www.webofscience.com/wos/author/record/NPI-9402-2025) from ORCID
